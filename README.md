# palindrome_test

---

## Getting started

### Prerequisites
At minimum you will need to have node installed.

### Getting the source
```
git clone https://github.com/pinhorox/palindrome_test.git
```

This will run the server with the default port of 3000.

## Development Environment
## 1. Install packages
```
npm install
```
## 2. Start tests
```
npm test
```
